Whether you’re considering a career in web development, looking to understand a bit of code to launch your own website, or you just want to learn something new – our Learn to Code Workshops are a great way to get introduced to the world of web development. In this 2-hour workshop we’ll cover:

  • Coding in HTML and CSS
  • Using a code editor (VScode) and code repository (Git)
  • Deploying your website using Github
  • Understanding how to add JavaScript to your website
